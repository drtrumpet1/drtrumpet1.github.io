Site Name

The site name will be Gem State Weather. This name is derived from the nickname for the state of Idaho, the Gem State.

Site Purpose

The purpose of this site is to inform Idaho citizens and visitors about the most current weather. There is weather information specific to three different areas in Idaho. It contains a photo gallery of some of the things that make Idaho beautiful. The site will also provide a form for users to submit severe weather incidents.
